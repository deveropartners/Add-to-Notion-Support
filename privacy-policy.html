Privacy Policy

Add to Notion for macOS and iOS — Effective April 2, 2026

Overview

Add to Notion is a lightweight client for the Notion API that lets you quickly add tasks and manage pages from your menu bar (macOS) or home screen (iOS). Your privacy is fundamental to how this app is built: it connects directly to Notion's servers with no intermediary servers, no analytics, and no tracking.

The app supports two authentication methods: OAuth (recommended), which lets you authorize the app through Notion's official sign-in flow, and manual API key, where you provide your own Notion internal integration token. In both cases, your credentials are stored locally on your device and all communication goes directly to Notion — no data passes through any server operated by the developer.

What Data the App Uses

Add to Notion handles only the data necessary to function:

  • OAuth Tokens — If you connect via OAuth, the app receives an access token and a refresh token from Notion after you authorize the app. These tokens allow the app to communicate with Notion on your behalf.
  • Notion API Key (alternative method) — If you choose manual setup instead of OAuth, the internal integration token you provide to authenticate with Notion.
  • Notion Workspace Data — Database names, page titles, property values, and other content fetched from your Notion workspace via the Notion API.
  • App Preferences — Your selected databases, favorites, filter settings, and display preferences.

Where Data Is Stored

All data remains on your device:

  • Authentication Credentials — Whether you use OAuth tokens or a manual API key, your credentials are stored in your device's Keychain using Apple's Security framework. Credentials are local to the device and are not synced via iCloud Keychain or any other service. On the same device, credentials are shared between the main app and its extensions (share extension) through a secure Keychain access group.
  • App Preferences — Your selected databases, favorites, filter settings, and display preferences are stored in a local file on your device. These are not synced or uploaded anywhere.
  • Cached Task Data — The app caches a summary of your Notion database items (titles, status, dates, and property values) in local files on your device. This cache enables the iOS widget to display your items and allows the app to show data while loading. Cached data is not synced or uploaded anywhere.
  • Recent Items — A list of recently added items (title, database, and timestamp) is stored locally for your convenience. This data is not synced or uploaded anywhere.

Network Connections

During normal use, the app communicates with one external service: Notion's API at api.notion.com. All requests use HTTPS and are authenticated with your OAuth token or API key.

If you connect via OAuth, a one-time redirect during the initial authorization passes through a static page hosted on GitHub Pages (deveropartners.github.io). This page performs a client-side redirect to pass the authorization code back to the app. It does not log, store, or process any data. After this initial authorization, the app exchanges the code directly with Notion's servers for access tokens. No subsequent requests go through GitHub Pages.

The app does not include any analytics, crash reporting, or telemetry services.

Data We Do Not Collect

To be explicit about what does not happen:

  • No data is sent to the developer or any third party.
  • No OAuth tokens or authorization codes are transmitted to or stored on any server operated by the developer.
  • No analytics, usage tracking, or event logging occurs.
  • No crash reports are transmitted.
  • No device information, identifiers, or location data is collected.
  • No advertising frameworks or tracking SDKs are included.
  • No accounts are created; the app uses either Notion's OAuth authorization flow or your own Notion integration token.

Third-Party Services

The app interacts with the following third-party services:

  • Notion (Notion's Privacy Policy) — All API requests and OAuth authorization are handled by Notion. Your use of the Notion API is subject to Notion's own terms and privacy policy.
  • GitHub Pages — Used solely as a static redirect during OAuth authorization. The redirect page contains only client-side JavaScript that passes the authorization code back to the app. No data is collected, logged, or stored by this page.

The developer of Add to Notion has no access to your Notion account or data.

Revoking Access

If you connected via OAuth, you can disconnect the app at any time by going to Notion Settings → My Connections, finding "Add to Notion," and clicking Disconnect. This immediately invalidates the app's access and refresh tokens. The app will prompt you to reconnect the next time you use it.

If you used a manual API key, you can revoke access by deleting or disabling the integration in your Notion workspace settings.

Children's Privacy

Add to Notion is not directed at children under the age of 13 and does not knowingly collect any personal information from children.
